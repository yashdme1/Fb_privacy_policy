Privacy Policy for FB Messenger Auto-Responder Bot

1. Introduction
This app connects a Facebook Page Messenger with OpenAI via Make.com to generate automated replies. It is intended to enhance user engagement through AI-generated responses.

2. Data Collection
We do not collect or store any personal information.
The only data used is the Page-Scoped ID (PSID) of the user, which is temporarily required to send a response back through Facebook Messenger.

3. Data Usage
The PSID is used in real-time within Make.com to:
- Route the user message to OpenAI for generating a reply
- Send the reply back to the user via Messenger

No other user data (such as name, email, or message content) is stored, saved, or reused after the automation completes.

4. Data Sharing
We do not share any data with third-party services other than:
- OpenAI (to generate responses based on the incoming message)
- Make.com (as the workflow automation platform)

5. Data Storage
We do not store any user messages or personal data. All actions are performed in real-time and data is discarded after response delivery.
